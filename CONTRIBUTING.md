# Contributing

All contributions, bug reports, bug fixes, documentation improvements, enhancements, and ideas are welcome.

## How to contribute
1. Fork this repository and create a feature branch.
2. Make your changes with clear, small commits.
3. Ensure scripts/tools still work (or note limitations if any).
4. Open a Pull Request with a clear title and description of changes and testing.

## Reporting issues
- Describe the problem, expected behavior, and steps to reproduce.
- Include environment details if relevant.
